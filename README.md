# Soccer Club Community Portal Project

## Overview
This Node.js-based web portal helps KQMF United connect with the community by showcasing matches, team members, and allowing fans to reach out.

## Technologies
- Node.js
- Express.js
- EJS
- HTML5
- CSS
- Git & GitHub 
- nodemon
